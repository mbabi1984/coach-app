# تطبيق التدريب — نسخة تجريبية (PWA + Supabase)

تطبيق تدريب عربي/إنجليزي، يشتغل على رابط (GitHub Pages) ويُضاف للشاشة الرئيسية كأنه تطبيق.
نسخة تجريبية لكوتش واحد ومتدربيه — بدون متاجر تطبيقات وبدون أي تكلفة.

## الملفات
- `index.html` — التطبيق كامل (واجهة المدرب + واجهة المتدرب).
- `config.js` — **الملف الوحيد اللي تعدّله**: رابط Supabase + مفتاح anon + رمز المدرب.
- `schema.sql` — تنسخه مرة وحدة في Supabase ليبني الجداول.
- `manifest.webmanifest` + `sw.js` + الأيقونات — عشان يصير قابل للتثبيت كـ PWA.

## التشغيل (باختصار)
1. أنشئ مشروع Supabase مجاني.
2. شغّل محتوى `schema.sql` في: Supabase ← SQL Editor.
3. من Settings ← API: انسخ **Project URL** و**anon public key** وحطهم في `config.js`، واختر رمز مدرب.
4. ارفع كل الملفات على مستودع GitHub وفعّل GitHub Pages (Branch: main / root).
5. افتح رابط الصفحة وجرّب. التفاصيل خطوة بخطوة في `chrome-prompt.md`.

## كيف يستخدمه الناس
- **المدرب**: يدخل برمزه ← يضيف متدرب (يطلع رمز دعوة) ← يبني برنامجه ← مكتبة التمارين ← محادثة.
- **المتدرب**: يدخل رمز الدعوة ← يسجّل بياناته ← يشوف تماريته ويسجّل أوزانه ← محادثة.
- **التثبيت على الجوال**: افتح الرابط ← قائمة المتصفّح ← "أضف إلى الشاشة الرئيسية".

## ملاحظة أمان
سياسات قاعدة البيانات مفتوحة لأغراض الاختبار الخاص فقط. لا تضع بيانات حسّاسة.
عند التوسّع لاحقاً نستبدلها بمصادقة حقيقية.
